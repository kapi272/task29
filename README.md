# CustamHooks

This is a simple React app made with Vite. It shows a grid of colors in the browser.

## How to run the project

Open a terminal and go into the project folder:

```bash
cd CustamHooks
```

Install the needed packages:

```bash
npm install
```

Start the app:

```bash
npm run dev
```

## How to build the project

Use this command when you want to make a production version:

```bash
npm run build
```

## How to publish

This project can be published with GitHub Pages.

1. Push the code to GitHub.
2. Open the repository settings on GitHub.
3. Go to Pages.
4. Select GitHub Actions as the source.

The app will deploy automatically after you push to `main`.
